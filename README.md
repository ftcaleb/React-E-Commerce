# React-E-Commerce
E-Commerce React Project
Overview

A collaborative group project built with React, featuring product browsing, cart management, address input, and an order success page. The goal was to create a clean, responsive shopping experience while practicing teamwork and shared development workflows.

Team Collaboration

Our team worked together using:

Git & GitHub for branching, pull requests, and reviews

Clear communication for assigning tasks and solving issues

Shared coding standards for consistency across components

Each member contributed to pages, components, Redux setup, and routing to complete the project efficiently.

Features

Product listing & Search

Cart functionality with Redux Toolkit

Address form & Order confirmation

Reusable components (Sidebar, ProductCard, Search)

Smooth navigation with React Router
